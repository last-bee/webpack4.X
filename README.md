##webpack的vue多页面的配置
-------
+ html-webpack-plugin 创建html 需要获取全部html文件同一配置
+ babel es6转换成es5
+ style css url less sass stylus css编译
+ postcss 兼容浏览器
+ html-widthimg 图片引入
+ webpack-dev-server 热加载
+ clean-webpack-plugin 清楚文件
+ copy-webpack-plugin copy资源
+ vue-loader vue的解析
+ vue-template-compile vue模板解析
